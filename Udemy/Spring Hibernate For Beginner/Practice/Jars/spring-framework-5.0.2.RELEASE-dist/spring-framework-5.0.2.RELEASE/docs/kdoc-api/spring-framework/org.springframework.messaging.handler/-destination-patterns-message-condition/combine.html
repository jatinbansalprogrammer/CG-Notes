combine

open fun combine(other: DestinationPatternsMessageCondition): DestinationPatternsMessageCondition

Returns a new instance with URL patterns from the current instance ("this") and the "other" instance as follows:

  • If there are patterns in both instances, combine the patterns in "this" with the patterns in "other" using org.springframework.util.PathMatcher#combine(String, String).
  • If only one instance has patterns, use them.
  • If neither instance has patterns, use an empty String (i.e. "").
