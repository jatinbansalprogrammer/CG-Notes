Parameters

StandardMethodMetadata(introspectedMethod: Method, nestedAnnotationsAsMap: Boolean)

Create a new StandardMethodMetadata wrapper for the given Method, providing the option to return any nested annotations or annotation arrays in the form of org.springframework.core.annotation.AnnotationAttributes instead of actual java.lang.annotation.Annotation instances.

Parameters

introspectedMethod - the Method to introspect

nestedAnnotationsAsMap - return nested annotations and annotation arrays as org.springframework.core.annotation.AnnotationAttributes for compatibility with ASM-based AnnotationMetadata implementations
