StaticListableBeanFactory()

Create a regular StaticListableBeanFactory, to be populated with singleton bean instances through #addBean calls.

StaticListableBeanFactory(beans: MutableMap<String, Any>)

Create a StaticListableBeanFactory wrapping the given Map.

Note that the given Map may be pre-populated with beans; or new, still allowing for beans to be registered via #addBean; or java.util.Collections#emptyMap() for a dummy factory which enforces operating against an empty set of beans.

Parameters

beans - a Map for holding this factory's beans, with the bean name String as key and the corresponding singleton object as value
