setExceptionMessage

open fun setExceptionMessage(exceptionMessage: String): Unit

Set the template used for method exception log messages. This template can contain any of the following placeholders:

  • $[targetClassName]
  • $[targetClassShortName]
  • $[argumentTypes]
  • $[arguments]
  • $[exception]
