setExitMessage

open fun setExitMessage(exitMessage: String): Unit

Set the template used for method exit log messages. This template can contain any of the following placeholders:

  • $[targetClassName]
  • $[targetClassShortName]
  • $[argumentTypes]
  • $[arguments]
  • $[returnValue]
  • $[invocationTime]
