setContainerFactoryBeanName

open fun setContainerFactoryBeanName(containerFactoryBeanName: String): Unit

Set the bean name of the JmsListenerContainerFactory to use in case a JmsListenerEndpoint is registered with a null container factory. Alternatively, the container factory instance can be registered directly: see #setContainerFactory(JmsListenerContainerFactory).
