setContainerFactory

open fun setContainerFactory(containerFactory: JmsListenerContainerFactory<*>): Unit

Set the JmsListenerContainerFactory to use in case a JmsListenerEndpoint is registered with a null container factory.

Alternatively, the bean name of the JmsListenerContainerFactory to use can be specified for a lazy lookup, see #setContainerFactoryBeanName.
