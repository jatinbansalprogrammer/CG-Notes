setDefaultDomain

open fun setDefaultDomain(defaultDomain: String): Unit

Set the default domain to be used by the MBeanServer, to be passed to MBeanServerFactory.createMBeanServer() or MBeanServerFactory.findMBeanServer().

Default is none.
